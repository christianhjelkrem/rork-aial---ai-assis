# Kultur i Ålesund — Events App for the Ålesund Region


## Concept

A beautiful coastal-themed events app that aggregates cultural happenings in the Ålesund region (Ålesund, Ørskog, Sula, Giske) into a single, easy-to-browse feed. Norwegian (Bokmål) interface. Data comes from your existing Supabase database.

Design inspiration: Ålesund's Art Nouveau architecture, the Norwegian coastline — ocean blues, seafoam greens, warm sand accents. Clean and modern with a distinctly Nordic-coastal feel.

---

## Features

- **Browse events** — A scrollable feed of upcoming cultural events, sorted by date
- **Search events** — Find events by name, location, or organizer
- **Filter by tags** — Tap tag chips to narrow down events (music, theater, film, etc.)
- **Event details** — Tap an event to see full info: description, time, location, price, and a link to buy tickets or learn more
- **Free events badge** — Easily spot free events with a visual indicator
- **Pull to refresh** — Swipe down to load the latest events from the database

---

## Design

- **Color palette:** Deep ocean blue (#1B3A4B) as primary, seafoam teal (#4ECDC4) as accent, warm sand (#F5E6D3) for backgrounds, white cards with subtle shadows
- **Cards:** Each event shown as a rich card with the event image, title, date/time, location, and price info
- **Tag chips:** Rounded, tinted pills for filtering by category
- **Typography:** Clean sans-serif, bold titles, lighter secondary text
- **Empty states:** Friendly illustrations when no events match your search
- **Micro-interactions:** Button press animations, smooth list transitions

---

## Screens

1. **Home / Events Feed** — The main screen with a search bar at top, horizontal tag filters, and a vertical list of event cards sorted by date
2. **Event Details** — Full-screen event detail with hero image, all event info (date, time, location, price, organizer, description), and an "Open link" button
3. **About** — Simple info screen about the app

---

## Tabs

- **Arrangementer** (Events) — The main feed
- **Om** (About) — Info about the app
